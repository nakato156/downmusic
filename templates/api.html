Api Down Music

Aparte de ser una web de descarga de música practica y segura también proveemos una api pública para que cualquiera pueda usarlo sin costo alguno.

A continuación se muestra un ejemplo de uso con JavaScript, en el cual se realiza un petición get a la ruta indicada. El tipo de devolución es un archivo mp3 y debe ser procesado de la forma en que usted crea conveniente.
